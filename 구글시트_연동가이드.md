# 구글 스프레드시트 연동 가이드 (데이터 자동 수집)

웹 브라우저의 양식에서 제출한 데이터가 구글 스프레드시트 1열(위에) 계속 쌓이게 하려면, Apps Script를 사용하여 API 엔드포인트를 만들어 주시면 됩니다.

주신 시트 링크: [구글 스프레드시트 링크](https://docs.google.com/spreadsheets/d/1_TgAvXAk1xM5Zxmre-kRbTygsG09qZHo5UVD6G5DMzk/edit?gid=394669539#gid=394669539)

### 🚀 적용 방법

1. 위 **구글 스프레드시트** 링크로 들어갑니다.
2. 상단 메뉴에서 **확장 프로그램** > **Apps Script**를 클릭합니다.
3. 열린 편집기 화면의 기존 코드를 지우고, 아래의 코드를 복사해서 붙여넣습니다.

```javascript
function doPost(e) {
  var sheet = SpreadsheetApp.openById("1_TgAvXAk1xM5Zxmre-kRbTygsG09qZHo5UVD6G5DMzk").getSheetByName("수탁 요청");
  var params = e.parameter;
  
  // 1열(최상단)에 계속 쌓이게 하기 위해 2번째 줄에 데이터 삽입
  var rowData = [
    new Date(),
    params.applicantName || "",
    params.applicantEmail || "",
    params.applicantPhone || "",
    params.companyName || "",
    params.department || "",
    params.ageGroup || "",
    params.position || "",
    params.attendeeCount || "",
    params.mainTask || "",
    params.trainingPurpose || "",
    params.budget || "",
    params.trainingPeriod || "",
    params.trainingDaysHours || "",
    params.trainingLocation || "",
    params.trainingType || "",
    params.pastExperience || "",
    params.pastExperienceDetails || "",
    params.bookPrinting || "",
    params.desiredSchedule || "",
    params.otherInquiries || ""
  ];

  sheet.insertRowAfter(1);
  sheet.getRange(2, 1, 1, rowData.length).setValues([rowData]);

  return ContentService.createTextOutput(JSON.stringify({"status":"success"})).setMimeType(ContentService.MimeType.JSON);
}
```

4. 상단 메뉴의 💾 **저장(디스켓 모양 아이콘)** 버튼을 누릅니다.
5. 우측 상단의 파란색 **[배포]** 버튼을 누르고 **[새 배포]**를 선택합니다.
6. 왼쪽 톱니바퀴를 눌러 **웹 앱**을 선택합니다.
7. 다음 설정을 맞춥니다:
   - **액세스할 수 있는 사용자**: `"모든 사용자"` (반드시 모든 사용자로 해야 외부 폼에서 접근이 가능합니다)
8. **[배포]** 버튼을 누릅니다. (권한 승인 창이 뜨면 구글 계정을 선택해 '고급 -> 안전하지 않음으로 이동' 후 허용해주시면 됩니다)
9. 완료되면 생성된 **웹 앱 URL**이 나옵니다. 이 URL을 복사합니다.
10. 이제 `c:\Users\KPC_User\Desktop\모집양식\script.js` 파일을 열고, 첫 번째 줄의 `GOOGLE_SCRIPT_URL` 변수 값을 방금 복사한 URL로 변경하고 다시 깃허브에 푸시하시면 웹 앱과 시트가 완전히 실시간 연동됩니다.
